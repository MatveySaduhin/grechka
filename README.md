# My nvim setup

## Requirements

- Neovim >= 0.12 (`vim.pack`)
- A terminal with true color support (optional)
- `git`)

## Installation

```bash
git clone https://github.com/MatveySaduhin/grechka.git ~/.dotfiles
mv ~/.config/nvim ~/.config/nvim.bak 2>/dev/null
ln -s ~/.dotfiles/nvim ~/.config/nvim
